# Descripción de la rama
Esta rama contiene ejemplos en Java que ilustran los principios SOLID en la carpeta `Principios_Solid`. Cada archivo presenta un principio:

- `DIP.java` — Principio de Inversión de Dependencias  
- `OCP.java` — Principio de Abierto/Cerrado  
- `LSP.java` — Principio de Sustitución de Liskov  
- `ISP.java` — Principio de Segregación de Interfaces  
- `SRP.java` — Principio de Responsabilidad Única
